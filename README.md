# Team Profile Generator

## Purpose
An application to allow a user to input employee data and generate an html page with all of the employees and their contact information displayed

## Built With
* HTML
* CSS
* Bootstrap
* JavaScript
* Node.js
* OOP
* Jest API
* Inquirer API
